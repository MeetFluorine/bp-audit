Upload base / system data

Upload your base file as-is — it auto-detects LocationCode as store, ItemNo as SKU, and SerialNo (falling back to IMEI if blank) as the serial to reconcile against. Extra columns like Description, MRP, LotNo etc. are ignored. This is locked in as the "expected" list before scanning starts.

Select store

Pick the store you're currently auditing. Scan serials with a barcode scanner (it types + presses Enter automatically) or type them manually, or upload an Excel of pre-scanned serials for this store. Note: if your uploaded file already has its own store/location column, that value is used instead of the dropdown — so a multi-store file will correctly split across stores, but a single-store file with a leftover store column from another export could get misattributed. Leave that column out if you want everything in the file counted under the selected store.
